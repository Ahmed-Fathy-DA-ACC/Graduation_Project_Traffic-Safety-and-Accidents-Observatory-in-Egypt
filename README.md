
# Traffic Safety and Accidents Observatory Analysis
## Project Description
This project focuses on analyzing traffic safety and road accidents in Egypt.  
The goal is to study accident data, extract insights, and support better decision-making to improve road safety.

---
## Technologies Used

### Statistical Analysis

* Applied statistical techniques to understand data behavior and uncover meaningful patterns.
* Performed distribution analysis, correlation analysis, and outlier detection.
* Used statistical insights to support data-driven decisions.

### Microsoft Excel

* Cleaned and prepared raw data for analysis.
* Handled missing values, duplicates, and data inconsistencies.
* Utilized advanced formulas, Pivot Tables, and charts for exploratory analysis.
* Leveraged Power Query for data extraction, transformation, and cleaning.
* Built data models and established relationships using Power Pivot.
* Developed interactive dashboards to present key insights effectively.
* Structured datasets to ensure data quality and consistency.

### SQL

* Extracted and manipulated data using complex SQL queries.
* Performed filtering, aggregation, and multi-table joins.
* Used subqueries and window functions to support analytical tasks.
* Designed relational data models to optimize reporting and analysis.
* Created structured datasets for reporting and visualization.

### Python

* Cleaned, transformed, and analyzed data using Pandas and NumPy.
* Conducted Exploratory Data Analysis (EDA) to identify trends and anomalies.
* Built visualizations with Matplotlib and Seaborn to communicate insights.
* Automated repetitive data processing tasks to improve efficiency.

### Tableau

* Designed interactive dashboards and visual reports.
* Created calculated fields, filters, and KPI indicators.
* Developed dynamic visualizations to communicate business insights effectively.
* Built and optimized data models to enhance analytical performance.

### Power BI

* Performed data transformation and ETL processes using Power Query.
* Built data models and established relationships between tables.
* Developed DAX measures and calculated columns for advanced analysis.
* Created interactive dashboards with drill-through and KPI tracking capabilities.
* Designed scalable data models to support efficient reporting.
* Delivered actionable insights through clear and effective visualizations.
